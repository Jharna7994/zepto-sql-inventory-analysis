# Zepto Inventory & Pricing Analysis (SQL)

This project analyzes Zepto's product inventory using SQL to understand
pricing, discounts, stock availability, and category-level performance.

## Analysis Includes
- Table creation and schema design
- Data exploration and null checks
- Data cleaning and price standardization
- Discount-based product analysis
- Stock availability insights
- Category-wise revenue estimation
- Price-per-gram value analysis

## Skills Used
- SQL (PostgreSQL)
- Aggregations & filtering
- CASE statements
- Data cleaning
- Business analysis

- `zepto_analysis.sql`
